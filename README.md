# LeetCode Solutions


## 问题列表

### 1. 两数之和
- **问题链接**: https://leetcode.com/problems/sum-of-two-integers/description/
- **解决方案**: `sum(num1, num2)` - 返回两个整数的和
- **时间复杂度**: O(1)
- **空间复杂度**: O(1)

### 2. 温度转换
- **问题链接**: https://leetcode.com/problems/convert-the-temperature/description/
- **解决方案**: `convertTemperature(celsius)` - 将摄氏温度转换为开尔文和华氏温度
- **时间复杂度**: O(1)
- **空间复杂度**: O(1)

### 3. 最小偶倍数
- **问题链接**: https://leetcode.com/problems/smallest-even-multiple/description/
- **解决方案**: `smallestEvenMultiple(n)` - 返回n的最小偶倍数
- **时间复杂度**: O(1)
- **空间复杂度**: O(1)

### 4. 根节点等于子节点之和
- **问题链接**: https://leetcode.com/problems/root-equals-sum-of-children/description/
- **解决方案**: `checkTree(root)` - 检查根节点的值是否等于左右子节点值之和
- **时间复杂度**: O(1)
- **空间复杂度**: O(1)

### 5. 数组异或操作
- **问题链接**: https://leetcode.com/problems/xor-operation-in-an-array/description/
- **解决方案**: `xorOperation(n, start)` - 执行数组异或操作
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)

### 6. 好数对数目
- **问题链接**: https://leetcode.com/problems/number-of-good-pairs/description/
- **解决方案**: `numIdenticalPairs(nums)` - 统计数组中好数对的数目
- **时间复杂度**: O(n²)
- **空间复杂度**: O(1)

### 7. 统计好三元组
- **问题链接**: https://leetcode.cn/problems/count-good-triplets/
- **解决方案**: `countGoodTriplets(arr, a, b, c)` - 统计满足条件的三元组数目
- **时间复杂度**: O(n³)
- **空间复杂度**: O(1)

### 8. 转换为小写字母
- **问题链接**: https://leetcode.com/problems/to-lower-case/description/
- **解决方案**: `toLowerCase(s)` - 将字符串转换为小写
- **时间复杂度**: O(n)
- **空间复杂度**: O(n)

### 9. 数字根
- **问题链接**: https://leetcode.com/problems/add-digits/description/
- **解决方案**: `addDigits(num)` - 计算数字的数字根
- **时间复杂度**: O(log n)
- **空间复杂度**: O(1)

### 10. 整数的各位积和之差
- **问题链接**: https://leetcode.com/problems/subtract-the-product-and-sum-of-digits-of-an-integer/description/
- **解决方案**: `subtractProductAndSum(n)` - 计算整数各位数字的积与和之差
- **时间复杂度**: O(log n)
- **空间复杂度**: O(1)

### 11. 2的幂
- **问题链接**: https://leetcode.com/problems/power-of-two/description/
- **解决方案**: `isPowerOfTwo(n)` - 判断整数是否是2的幂
- **时间复杂度**: O(1)
- **空间复杂度**: O(1)

### 12. 3的幂
- **问题链接**: https://leetcode.com/problems/power-of-three/description/
- **解决方案**: `isPowerOfThree(n)` - 判断整数是否是3的幂
- **时间复杂度**: O(log₃n)
- **空间复杂度**: O(1)

### 13. 丑数
- **问题链接**: https://leetcode.com/problems/ugly-number/description/
- **解决方案**: `isUgly(n)` - 判断整数是否是丑数
- **时间复杂度**: O(log n)
- **空间复杂度**: O(1)

### 14. 重新排列数组
- **问题链接**: https://leetcode.com/problems/shuffle-the-array/description/
- **解决方案**: `shuffle(nums, n)` - 按照指定规则重新排列数组
- **时间复杂度**: O(n)
- **空间复杂度**: O(n)

### 15. 转置矩阵
- **问题链接**: https://leetcode.com/problems/transpose-matrix/description/
- **解决方案**: `transpose(matrix)` - 返回矩阵的转置
- **时间复杂度**: O(m×n)
- **空间复杂度**: O(m×n)

### 16. 分割字符串的最大得分
- **问题链接**: https://leetcode.com/problems/maximum-score-after-splitting-a-string/description/
- **解决方案**: `maxScore(s)` - 计算分割字符串后能得到的最大得分
- **时间复杂度**: O(n²)
- **空间复杂度**: O(n)

### 17. 统计范围内的元音字符串
- **问题链接**: https://leetcode.com/problems/count-vowel-strings-in-ranges/description/
- **解决方案**: `vowelStrings(words, left, right)` - 统计指定范围内首尾都是元音的字符串数目
- **时间复杂度**: O(n)
- **空间复杂度**: O(1)

## 使用说明
